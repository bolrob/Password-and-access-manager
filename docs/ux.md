# UX-концепция

## Персоны  
### Персона 1: Алексей, 30 лет, IT-специалист  
- Цели: быстро сохранять и находить рабочие доступы (серверы, админки), не запоминать пароли.  
- Важно в первые 30 сек: увидеть список последних использованных записей, кнопку «Добавить» и поиск.  

## User Stories  
1. Как зарегистрированный пользователь, хочу добавить новую учётную запись, чтобы не помнить пароль вручную.  
   - Шаги: Нажать «+», ввести название/логин, сгенерировать пароль, выбрать категорию, сохранить.  
   - Done-критерий: запись появляется в общем списке, пароль скрыт звёздочками, видна оценка надёжности.  
2. Как пользователь, хочу оценить надёжность своих паролей, чтобы понимать, какие пора менять.  
   - Шаги: Открыть vault, видеть цветовые индикаторы (красный/жёлтый/зелёный), отсортировать по слабым.  
   - Done-критерий: слабые пароли подсвечены, есть подсказка «пора заменить».  
3. Как пользователь, хочу установить напоминание о смене пароля через N дней, чтобы не забыть обновить важные доступы.  
   - Шаги: При создании/редактировании задать срок, получать отметку «просрочен».  
   - Done-критерий: в списке рядом с записью отображается статус (просрочен, скоро).  

## Дизайн-система (Taiga UI 4)  
- Используемые компоненты: Input, Password, Select, Checkbox, Toggle, Tag, Button, Card, Table, Loader, Notifications, Dialogs, Scroller, Tabs, Badge, Tooltip.  
- Кастомные компоненты: PasswordStrengthMeter, PasswordHistoryTimeline, ExportDialog.  

## Приоритеты  
- Первый экран: список записей (vault) с поиском и кнопкой добавления.  
- Вторично: страница отдельной записи с деталями, историей, возможностью редактирования.
